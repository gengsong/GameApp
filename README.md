#此项目演示怎样集成安装SDK
##支持如下功能：
###
* 此Demo用演示集成安装SDK，具体步骤如下：
* 1、MainActivity 中初始化完成后调用installAPKs方法
* 2、res-xml 实现filepaths.xml
* 3、AndroidManifest.xml中定义 provider,并设置权限uses-permission
* 4、assets文件夹放置要安装的apk
